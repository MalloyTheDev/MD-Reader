MD Reader

A fast, private, offline-first Markdown reader, library & editor for Windows - built for real technical, scientific, and study work.

⬇ Download for Windows View on GitHub

Free & open source (MIT) · works fully offline · installer is unsigned (install notes)

Built for technical Markdown

Most Markdown apps choke on the hard stuff. MD Reader renders KaTeX math (with per-equation copy/expand), Mermaid diagrams (zoom · pan · fullscreen · export), safe declarative charts, GFM tables, and science/engineering callouts - then lets you search it all with operators like has:math and tag:physics, link notes with [[wiki-links]], and export to HTML/Word with everything rendered.

Everything in your notes, in one place

📐 Math & science

KaTeX math with per-equation copy/expand, Mermaid diagrams, and safe declarative charts.

📚 A real library

A managed vault, folders, import, recent-folder navigation, tags, and a graph view.

🔎 Powerful search

Operators like tag:, has:math, content: with previews.

✍️ Editor & templates

Live editor, slash menu, CSV↔table, and 15 curated note templates.

🧠 Study tools

Highlights & notes, flashcards with spaced repetition, and a document-info panel.

📤 Export anywhere

HTML & Word export with math, diagrams, and charts fully rendered - plus print/PDF.

🔒 Private by default

MD Reader runs entirely on your machine. File access is sandboxed to the folder you open, remote images are blocked by default, deletes go to the Recycle Bin, and Mermaid/charts render with no code execution. AI features are optional and bring-your-own-key - your key is stored encrypted in the OS keychain and only the request you trigger ever leaves your computer. See the security model.
